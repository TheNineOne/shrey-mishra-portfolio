# Portfolio & Artist Page

This is a premium portfolio website for Shrey Mishra, built with Next.js, Framer Motion, and Tailwind CSS.

## Features
- **Professional Portfolio**: Highlights development skills (Java, Spring Boot).
- **NineOne Artist Page**: A cinematic high-energy page for the content creator / artist persona.
- **Modern UI**: Full of interactive animations and smooth transitions.
- **Dark Mode**: Premium deep-navy aesthetics.

## Deployment
Deployed on Vercel.
